**# OOP PHP Product Management System

## 📌 Project Description
This is a simple Product Management System built using PHP and Object-Oriented Programming (OOP) concepts.  
The project demonstrates how to handle product data and perform basic business logic operations such as discount and tax calculations.

---

## 🚀 Features
- Create product objects with detailed information
- Store product data (name, price, brand, image, description, tax)
- Calculate price after discount
- Calculate price after tax
- Display products in a responsive UI using Bootstrap
- Apply OOP principles (Classes, Objects, Constructor, Encapsulation, Methods)

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- PHP Classes & Objects
- Constructor Method
- Encapsulation
- Basic arithmetic operations (discount & tax)
- Dynamic content rendering with PHP
- Bootstrap for UI design

---

## 🛠️ Technologies Used
- PHP
- HTML
- CSS (Bootstrap 5)
- XAMPP (Local Server)
